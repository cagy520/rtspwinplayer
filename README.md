# rtspwinplayer
WinplayerForRtsp

本项目来源于 https://github.com/i-TechSoftware/RtspClientSharpCore.git 在此基础上进行修改
有一部分是C语言的库，记得把原始项目拉下来

项目功能：
在winform接收播放rtsp视频流

检测视频变化相似度

推送MQTT消息报警
